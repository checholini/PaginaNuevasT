¿Qué es la materia de nuevas tecnologías?

Nuevas Tecnologias es una clase enfocada en el uso y administración de nuevas tecnologías en el desarrollo web.

Los objetivos de la clase son:

  • Desarrollo y manejo de aplicaciones web.
  • El uso de modelos para la construcción de BackEnd y FrontEnd.
  • El uso de Bases de datos relacionales y no-relacionales en aplicaciones webs.

Tematicas de la materia.

Los temas son:

  • Patrón MVC
  • Servidores
  • Modelo-JPA
  • BD Relacionales y no relacionales
  • Controladores
  • Servicios REST, EJB
  • Patrones (DAO, DTO)
  • Vista
  • Python/Django
  • NodeJs

Tematicas
Patrón MVC

Patrón de arquitectura de software, se divide en: el modelo, la vista y el controlador. Y se utiliza en la creación de aplicaciones web separando los datos y la lógica de negocio.

Servidores

Más en concreto Servidores de aplicaciones, los cuales nos ayudan a la distribución de recursos para despliegue de aplicaciones web..


Modelo-JPA

El JPA (Java Persistence API) es un modelo para mapear bases de datos relacionales en Java, aplicaciones web y aplicaciones clientes, su principal función es el mapeo de relación entre entidades Java y tablas de la base de datos.

BD Relacionales y no relacionales

En este tema veremos más a profundidad el uso de bases de datos (Tanto relacionales como no relacionales) en aplicaciones web.


Controladores

La capa controladora del modelo MVC es la encargada de cargar capa modelo y invocar capa vista, Esto para que no se tenga un interacción directa entre la vista y el modelo.

Servicios REST, EJB

La API REST es uno de los servicios para la obtención de datos o generar operaciones sobre esos datos, La API EJB es un servicio intermediario para el acceso de datos.


Patrones (DAO, DTO)

El JPA (Java Persistence API) es un modelo para mapear bases de datos relacionales en Java, aplicaciones web y aplicaciones clientes, su principal función es el mapeo de relación entre entidades Java y tablas de la base de datos.

Vista

La vista es toda la interfaz con la cual interactúa el usuario, en la cual organizaremos la distribución de datos.


Python/Django

Veremos cómo se utiliza el lenguaje Python en la creación de aplicaciones web y como transportar todo lo visto de esta materia en la framework Django, el cual utiliza como principal lenguaje Python.

NodeJs

código abierto de JavaScript que está diseñado para generar aplicaciones web de forma altamente optimizada.
